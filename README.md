# 🌾 Agentic RAG — Digestats Regulatory Compliance

LangGraph-based agentic RAG system answering questions about French and European regulations on agricultural digestates (CDC DIG, NF U 44-051, EU Regulation 2019/1009).

🚧 README in progress — full documentation coming soon.

## Stack

LangGraph · BGE-M3 + BM25 hybrid retrieval · Qdrant · Gemini 2.5 Flash · Streamlit · Cloud Run · Langfuse · DeepEval

## Status

Active development — production deployment on Google Cloud Run.
